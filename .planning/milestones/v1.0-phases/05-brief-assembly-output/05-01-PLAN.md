  <behavior>
    generateBrief overall:
    - Takes BriefInput (extraction, exportResult, projectPath, fileName, figmaUrl)
    - Returns BriefResult { markdown, charCount, estimatedTokens, stats }
    - Markdown contains 6 sections in locked order: metadata, preview, layout tree, design tokens, components, assets
    - Each section has a ## heading
    - Empty sections (e.g., no shadows) are omitted entirely

    Metadata section:
    - "# Design Brief" as h1
    - File name, frame name (first root node name), extraction date (YYYY-MM-DD), Figma URL
    - All as bold label + value pairs

    Preview section:
    - "## Preview" heading
    - Markdown image link to preview path (project-relative)
    - Omitted if previewPath is empty

    Layout tree section:
    - "## Layout Tree" heading
    - Indented tree with 2-space indentation per depth level
    - Each line: Type 'Name' (auto-layout props) WxH
    - INSTANCE nodes: Instance "ComponentName" (variant props if any) WxH
    - INSTANCE with repeatCount > 1: Instance "ComponentName" x5 (repeated) WxH
    - TEXT nodes: Text 'content up to 60 chars...' (FontFamily Size/Weight)
    - Auto-layout props: flexDirection, gap (skip 0), justify (skip flex-start), align (skip flex-start), padding (skip all-zero), wrap (only show if wrap)
    - Compact padding: uniform -> "padding: N", symmetric -> "padding: N N", else 4 values
    - Positioning: ABSOLUTE nodes get [absolute] suffix
    - Hidden nodes (visible: false) are SKIPPED entirely
    - Full tree depth (no cap)

    Design tokens section:
    - "## Design Tokens" heading
    - Subsection "### Colors" with markdown table: Name | Value | Usage
    - Subsection "### Gradients" (only if gradients exist) with table: Name | Value | Usage
    - Subsection "### Typography" with table: Name | Font | Size | Weight | Line Height
    - Subsection "### Spacing" with table: Value | Sources | Usage
    - Subsection "### Borders" (only if borders exist) with table: Name | Radius | Stroke | Usage
    - Subsection "### Shadows" (only if shadows exist) with table: Name | Type | Value | Usage
    - Shadow value format: "Xpx Ypx Bpx Spx color" (offset, blur, spread, color)
    - Border stroke format: "Wpx color" or "--" if no stroke

    Components section:
    - "## Components" heading
    - Markdown table: Component | Source | Variants | Usage
    - Variant column: comma-separated "key: value" pairs, or "--" if none
    - Omit section if no components

    Assets section:
    - "## Assets" heading
    - Markdown table: File | Type | Path
    - Preview row first (type "Preview")
    - Then each asset with type based on extension (.svg -> "SVG", .png -> "PNG")
    - All paths project-relative (strip projectPath + "/" prefix)
    - Omit section if no assets and no preview

    Token estimation:
    - estimateTokens(markdown: string): number -- Math.ceil(markdown.length / 4)
    - TOKEN_WARNING_THRESHOLD = 12000

    BriefStats:
    - nodeCount: extraction.nodeCount
    - colorCount: tokens.colors.length
    - fontCount: tokens.typography.length
    - assetCount: exportResult.assets.length
    - estimatedTokens: from estimation

    Internal section builder functions (not exported -- keep as module-private):

    - `buildMetadataSection(input: BriefInput): string` -- h1 "# Design Brief", bold field/value pairs for file, frame (first root node name), date (new Date().toISOString().slice(0,10)), URL.

    - `buildPreviewSection(previewPath: string, projectPath: string): string` -- "## Preview\n\n![Preview](relative-path)" or empty string if no previewPath.

    - `buildLayoutTree(nodes: LayoutNode[], indent: number): string` -- Recursive. For each node:
      - Skip if `node.visible === false`
      - Call `renderNodeLine(node, indent)` to get the line
      - Recurse into children (unless componentRef -- INSTANCE nodes are leaf)
      - Join with newlines

    - `renderNodeLine(node: LayoutNode, depth: number): string` -- Build line parts:
      - 2-space indent per depth
      - Type/name/component/text label
      - Auto-layout props (skip defaults: justify flex-start, align flex-start, gap 0, all-zero padding, nowrap)
      - Compact padding formatting (uniform, symmetric, or 4-value)
      - Dimensions WxH (round to integer)
      - [absolute] suffix if positioning === 'ABSOLUTE'

    - `buildDesignTokensSection(tokens: DesignTokens): string` -- "## Design Tokens" heading, then subsections for each token type. Only include subsections that have data. Tables use markdown pipe format with header row and separator.

    - `buildComponentsSection(components: ComponentInventoryEntry[]): string` -- "## Components" table or empty string.

    - `buildAssetsSection(previewPath: string, assets: ExportResult['assets'], projectPath: string): string` -- "## Assets" table with relative paths or empty string.

    - `toRelativePath(absolutePath: string, projectPath: string): string` -- Strip projectPath prefix. If path starts with projectPath + '/', return the rest. Otherwise return as-is.

    Compose in generateBrief:
    ```typescript
    const sections = [
      buildMetadataSection(input),
      buildPreviewSection(...),
      buildLayoutTree(...),
      buildDesignTokensSection(...),
      buildComponentsSection(...),
      buildAssetsSection(...),
    ].filter(Boolean);
    const markdown = sections.join('\n\n');
    ```

    Compute stats and return BriefResult.
