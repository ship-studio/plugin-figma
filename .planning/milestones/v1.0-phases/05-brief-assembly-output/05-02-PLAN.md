  <action>
    Create `src/brief/io.ts` with two async functions for shell-based I/O.

    **CRITICAL: Shell metacharacter safety.** Markdown content contains single quotes, double quotes, backticks, dollar signs, parentheses, pipes, and every other shell metacharacter. NEVER interpolate raw markdown into shell commands. Use base64 encoding to safely pass content through shell.

    ```typescript
    import type { Shell } from '../types';

    /**
     * Save brief markdown to .shipstudio/brief.md in the project directory.
     * Uses base64 encoding to avoid shell metacharacter issues.
     */
    export async function saveBrief(
      shell: Shell,
      projectPath: string,
      markdown: string,
    ): Promise<void> {
      const briefDir = `${projectPath}/.shipstudio`;
      const briefPath = `${briefDir}/brief.md`;

      // Ensure directory exists
      await shell.exec('mkdir', ['-p', briefDir]);

      // Encode to base64 to safely pass through shell.
      // btoa only handles Latin1, so encode UTF-8 first via encodeURIComponent + unescape.
      const encoded = btoa(unescape(encodeURIComponent(markdown)));
      const result = await shell.exec('bash', [
        '-c',
        `echo '${encoded}' | base64 -d > '${briefPath}'`,
      ]);

      if (result.exit_code !== 0) {
        throw new Error(`Failed to save brief: ${result.stderr}`);
      }
    }

    /**
     * Copy markdown to macOS clipboard via pbcopy.
     * Uses base64 encoding for safety.
     */
    export async function copyToClipboard(
      shell: Shell,
      markdown: string,
    ): Promise<void> {
      const encoded = btoa(unescape(encodeURIComponent(markdown)));
      const result = await shell.exec('bash', [
        '-c',
        `echo '${encoded}' | base64 -d | pbcopy`,
      ]);

      if (result.exit_code !== 0) {
        throw new Error(`Clipboard copy failed: ${result.stderr}`);
      }
    }
    ```

    **Note on base64 single quotes:** If the base64 string itself contains a single quote (it won't -- base64 output is [A-Za-z0-9+/=] only), this would break. But base64 encoding is safe by definition. However, if `btoa` is not available in the Ship Studio runtime, fall back to an alternative:
    - Write markdown to a temp file first: `shell.exec('bash', ['-c', 'cat > /tmp/shipstudio-brief.md'])` -- but this also has escaping issues.
    - Best fallback: Use `shell.exec('bash', ['-c', 'base64 -d <<< "..." | pbcopy'])` with heredoc.
    - For now, use `btoa` as the primary approach per research recommendation. If it fails at runtime, the error will be caught by the caller.
