    **Step 1: Create src/layout/types.ts**
    Define LayoutNode, AutoLayoutProps, ComponentRef, ExtractionResult interfaces exactly as described in behavior section. These are the normalized output types consumed by Phase 3+.

    Export `LayoutConstraint` re-exported from `@figma/rest-api-spec` for convenience.

    **Step 2: Create src/layout/flexbox-map.ts**
    Implement `mapToFlexbox(frame: any): AutoLayoutProps`:
    - Map `layoutMode` HORIZONTAL -> 'row', VERTICAL -> 'column'
    - Map `primaryAxisAlignItems`: MIN -> 'flex-start', CENTER -> 'center', MAX -> 'flex-end', SPACE_BETWEEN -> 'space-between'
    - Map `counterAxisAlignItems`: MIN -> 'flex-start', CENTER -> 'center', MAX -> 'flex-end', BASELINE -> 'baseline'
    - Read `itemSpacing` as `gap`, defaulting to 0
    - Read `paddingTop/Right/Bottom/Left`, defaulting each to 0
    - Map `layoutWrap`: 'WRAP' -> 'wrap', otherwise -> 'nowrap'
    - When wrapping, include `rowGap` from `counterAxisSpacing`

    Implement `describeSizing(mode: string | undefined, resolvedPx: number | undefined): string`:
    - 'HUG' -> `hug (${resolvedPx}px)` or 'hug' if no resolved value
    - 'FILL' -> 'fill'
    - 'FIXED' or default -> `${resolvedPx}px` or '0px'

    **Step 3: Create src/layout/normalize.ts**
    Implement `normalizeNode(figmaNode: any, components: Record<string, any>, depth: number): LayoutNode | null`:
    - Return null for SLICE type nodes
    - Build base LayoutNode with id, name, type, visible (default true when undefined -- use `node.visible !== false`)
    - Read dimensions from `absoluteBoundingBox` (null-check first, fall back to `size` property)
    - Read `layoutSizingHorizontal/Vertical` as widthMode/heightMode
    - Read `layoutPositioning` for absolute positioning flag (LYOT-05)
    - Read `constraints`, `minWidth/maxWidth/minHeight/maxHeight`, `preserveRatio`
    - Check for auto-layout: if `layoutMode` exists and is not 'NONE', call `mapToFlexbox`
    - Handle TEXT nodes: read `characters` as `textContent`
    - Handle INSTANCE nodes: call `buildComponentRef()` to build componentRef, do NOT recurse into children
    - Handle BOOLEAN_OPERATION nodes: treat as leaf, do NOT recurse into children
    - For all other nodes with `children` array: recurse, filter nulls, then call `deduplicateChildren`

    Implement `buildComponentRef(instance: any, components: Record<string, any>): ComponentRef`:
    - Look up componentId in components map
    - Set componentName from map (fallback to instance.name)
    - Set description from map (if not empty string)
    - Set isRemote and source: remote=true -> 'library', remote=false -> 'local'
    - Extract variant properties from `componentProperties`: include VARIANT, BOOLEAN, TEXT types
    - Include `overrides` array if present

    Implement `countNodes(node: any): number`:
    - Count 1 for current node, recurse into children array if present

    Implement `deduplicateChildren(children: LayoutNode[]): LayoutNode[]`:
    - Build fingerprint for each child with componentRef (componentId + JSON of variantProperties)
    - Track counts per fingerprint
    - For groups of 3+ identical instances: keep first, set repeatCount, skip rest
    - Preserve original array order for non-deduplicated items

    Implement `normalizeTree(rootNodes: any[], components: Record<string, any>): ExtractionResult`:
    - Count total nodes across all roots using countNodes
    - Call normalizeNode on each root, filter nulls
    - Return { rootNodes: normalized, nodeCount, truncated: false }
