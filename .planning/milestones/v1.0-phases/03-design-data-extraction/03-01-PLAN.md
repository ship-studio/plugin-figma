Figma node properties are accessed via 'any' type with 'in' checks. Pattern: `if ('fills' in node && Array.isArray(node.fills)) { ... }`

From @figma/rest-api-spec (key types for reference):
- Paint: union of SolidPaint | GradientPaint | ImagePaint (each has `type`, `visible?`, `opacity?`)
- SolidPaint: `{ type: 'SOLID', color: RGBA, visible?, opacity? }`
- GradientPaint: `{ type: 'GRADIENT_LINEAR'|'GRADIENT_RADIAL'|'GRADIENT_ANGULAR'|'GRADIENT_DIAMOND', gradientHandlePositions: Vector[], gradientStops: ColorStop[], visible?, opacity? }`
- ImagePaint: `{ type: 'IMAGE', scaleMode: string, imageRef?: string, visible?, opacity? }`
- RGBA: `{ r: number, g: number, b: number, a: number }` (all 0-1 floats)
- Effect: `{ type: 'DROP_SHADOW'|'INNER_SHADOW'|..., visible: boolean, color: RGBA, offset: Vector, radius: number, spread?: number }`
- TypeStyle: `{ fontFamily, fontSize, fontWeight, lineHeightPx, letterSpacing, ... }`
- ColorStop: `{ position: number, color: RGBA }`
